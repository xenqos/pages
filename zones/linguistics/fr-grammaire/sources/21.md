# Relatif

## Formes simples
[|
|%[r|qui ]|
|%[r|que ]|
|%[r|quoi]|
|%[r|dont]|
|%[r|où  ]|
|]

[!
%[r|qui] - подлежащие, антецедент - одушевлёный/неодушевлёный
%[r|quе] - прямое дополнение, антецедент - одушевлёный/неодушевлёный
%[r|quoi] - неопределённое, антецедент - %[r|ce]
%[r|dont] - %[r|de qui, de quoi, duquel, de laquelle, desquels, desquelles]
%[r|ou] - обстоятельство места/времени, неодушевлённые
!]

## Formes complexes

[|
^s.           ^p.       ^
|%[r|lequel  ]|lequels  |
|%[r|laquelle]|laquelles|
|]

[!
%[r|lequel] - прямое дополнение с предлогом, антецедент - одушевлёные/неодушевлёные
!]

[!
%[r|à ] + %[r|lequel    ] = %[r|auquel    ]
%[r|à ] + %[r|lesquels  ] = %[r|auxquel   ]
%[r|à ] + %[r|lesquelles] = %[r|auxquelles]
%[r|de] + %[r|lequel    ] = %[r|duquel    ]
%[r|de] + %[r|lesquels  ] = %[r|desquels  ]
%[r|de] + %[r|lesquelles] = %[r|desquelles]
!]
