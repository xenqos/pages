# Untrennbare Präfixe

__ Untrennbare Vorsilben __

%[r|be]-
> особых смысловых оттенков нет
> грамматически всегда даёт аккузатив + прямое дополнение
>> %[r|besuchen] посещать

%[r|er]-
> особых смысловых оттенков нет
> грамматически всегда даёт аккузатив + прямое дополнение
>> %[r|erzählen] рассказывать

%[r|ge]-:
> обычно будет означать моментальное событие
> отмечает начало или конец какого-то процесса
>> %[r|gefallen] нравиться
>> %[r|gestehen] признавать
>> %[r|gedenken] вспоминать

%[r|ver]-
> %[b|перемещение]
> %[b|удаление]
> %[b|угасание]
>> %[r|versetzen] пересаживать
>> %[r|verstellen] переставлять
>> %[r|verschicken] посылать

%[r|zer]-
> %[b|разделение]
> %[b|раздробление]
> %[b|распад]
>> %[r|zerbrechen] разбивать
>> %[r|zerfallen] распадаться
>> %[r|zerschneiden] разрезать

%[r|ent]-
> %[b|удаление]
> %[b|отделение]
> %[b|начало процесса]
>> %[r|entdecken] открывать
>> %[r|entspannen] расслабиться
>> %[r|entsprechen] соответствовать

[!
%[r|emp]- производная от %[r|ent]-
!]

%[r|emp]-
>> %[r|empfinden] ощущать
>> %[r|empfehlen] рекомендовать
>> %[r|empfangen] принимать

%[r|miss]-
> %[b|недостаточность]
> %[b|несовершенство]
> %[b|неправильность]
>> %[r|misslingen] не удаться
>> %[r|missverstehen] неправильно] понимать
>> %[r|missbrauchen] неправильно] злоупотреблять
