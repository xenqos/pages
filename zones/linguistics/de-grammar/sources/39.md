# Wechselpräfixe

__ Trennbaren und untrennbaren Vorsilben __

%[r|durch]-
> движение через, завершение, проникновение
>> %[r|durchgehen] проходить
>> %[r|durchlesen] читать
>> %[r|durchdringen] проникать

%[r|über]-
> движение поверх, перемену места, превышение уровня
>> %[r|überlegen] класть поверх
>> %[r|überfliegen] перелетать
>> %[r|überfließen] переливаться через край

%[r|um]-
> движение вокруг, изменение, окружение
>> %[r|umgehen] ходить вокруг
>> %[r|umstellen] переставлять
>> %[r|umbauen] перестраивать

%[r|unter]-
> движение под, размещение, подчинение
>> %[r|unterschreiben] подписывать
>> %[r|unterteilen] разделять
>> %[r|unterordnen] подчинять

%[r|wider]-
> противодействие, возражение, ответное действие
>> %[r|widerstehen] устоять
>> %[r|widersprechen] возражать
>> %[r|widerhallen] резонировать, отзываться

%[r|wieder]-
> возвращение, восстановление, повторяющееся действие
>> %[r|wiederholen] повторить
>> %[r|wiedergeben] возвращать
>> %[r|wiedersehen] вновь увидеть

%[r|voll]-
> полностью, сполна
>> %[r|vollenden] завершать
>> %[r|vollziehen] исполнять
>> %[r|vollbringen] осуществлять

%[r|hinter]-
> движения назад, размещение позади, тайное действие
>> %[r|hintergehen] обманывать
>> %[r|hinterlegen] отдавать на хранение
>> %[r|hinterbringen] доносить, тайно сообщать
