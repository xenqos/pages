<h1>Mesh Current Method (Loop Current Method)</h1>
<img src="/uploads/thumbnails/mesh_current_method_answer_card_thumbnail.jpg" />
<h3 class="text-lg lg:text-2xl font-bold mb-3">What is the Mesh Current Method for Network Analysis?</h3>
<p><p>The mesh current method is a network analysis technique where mesh (or loop) current directions are assigned arbitrarily, and then Kirchhoff&rsquo;s voltage law (KVL) and Ohm&rsquo;s law are applied systematically to solve for the unknown currents and voltages.</p></p>
<p>This page provides a step-by-step introduction to using the mesh current method (also known as the loop current method) for analyzing electrical circuits. The mesh current method uses simultaneous equations, <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-6/kirchhoffs-voltage-law-kvl/" target="_blank">Kirchhoff&rsquo;s voltage law (KVL)</a>, and <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-2/voltage-current-resistance-relate/" target="_blank">Ohm&rsquo;s law</a> to determine unknown <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-1/voltage-current/" target="_blank">currents</a> in a network.</p>
<p>The mesh current method is quite similar to the branch current method. However, it does not use <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-6/kirchhoffs-current-law-kcl/" target="_blank">Kirchhoff&rsquo;s current law (KCL)</a>, and it is usually able to solve a circuit with fewer unknown variables and fewer simultaneous equations.</p>
<p>&nbsp;</p>

<p>Diving into the mesh current method, it&rsquo;s important to note that we&rsquo;ll be using the same example circuit (Figure 1) that we&rsquo;ve been using to introduce other <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-10/what-is-network-analysis/" target="_blank">network analysis methods</a>:</p>

<h4>Step 1: Identify and Label the Current Loops</h4>
<p>The first step in the mesh current method is identifying and labeling the current &ldquo;loops&rdquo; within the circuit. To do this, we must find at least one loop current passing through every component in the circuit.&nbsp;</p>
<p>In our example circuit (Figure 2), the loop formed by B<sub>1</sub>, R<sub>1</sub>, and R<sub>2</sub> will be the first loop, while the loop formed by B<sub>2</sub>, R<sub>2</sub>, and R<sub>3</sub> will be the second.</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_2._Step_1-_Identify_and_label_the_current_loops_.jpg" /></p>
<h5><em><strong>Figure 2.</strong>&nbsp;Identify and label the current loops.</em></h5>
<p>&nbsp;</p>
<p>The strangest part of the mesh current method is envisioning circulating currents in each of the loops. In fact, this method gets its name from the idea of these currents meshing together between loops like sets of spinning gears.</p>
<p>The choice of each current loop&rsquo;s direction is entirely arbitrary; however, the resulting equations are often easier to solve if the currents are going in the same direction through components with multiple current loops. For example, note how currents I<sub>1</sub> and I<sub>2</sub> both flow &ldquo;down&rdquo; through resistor R<sub>2</sub>, where they &ldquo;mesh&rdquo; or intersect. If the assumed direction of a mesh current is wrong, the answer for that current will have a negative value.</p>
<p>&nbsp;</p>
<h4>Step 2: Label the Voltage Drop Polarities</h4>
<p>The next step is to label all <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-2/polarity-voltage-drops/" target="_blank">voltage drop polarities</a> across resistors according to the assumed directions of the mesh currents, as shown in Figure 3.</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_3._Label_the_voltage_drop_polarities_.jpg" /></p>
<h5><em><strong>Figure 3.</strong> Label the voltage drop polarities.</em></h5>
<p>&nbsp;</p>
<p>Remember that the &ldquo;upstream&rdquo; end of a resistor will always be positive, and the &ldquo;downstream&rdquo; end of a resistor negative. This situation happens because the resistor is a load that drops voltage when current flow through it.&nbsp;</p>
<p>As a note, battery polarities are dictated by their symbol orientations in the diagram and may or may not &ldquo;agree&rdquo; with the resistor polarities and assumed loop current directions.</p>
<p>&nbsp;</p>
<h4>Step 3: Apply Kirchhoff&rsquo;s Voltage Law to Each Loop</h4>
<p>Using Kirchhoff&rsquo;s voltage law, we can now step around each of these loops, generating equations representative of the component <a href="https://www.allaboutcircuits.com/technical-articles/what-is-a-voltage-drop-in-an-electric-circuit/" target="_blank">voltage drops</a> and polarities. As with the branch current method, we will denote a resistor&rsquo;s voltage drop as the product of the <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-1/resistance/" target="_blank">resistance</a> (in ohms) and its respective mesh current (that quantity being unknown at this point). Where two currents mesh together, we will write that term in the equation, with the resistor current being the sum of the two meshing currents.</p>
<p>The starting point for tracing the voltage drops around each loop is arbitrary, the same as the direction we trace. Beginning with the left loop of the circuit, let&rsquo;s start at the lower-left corner and trace clockwise, counting polarity as if we had a voltmeter in hand, red lead on the point ahead, and black lead on the point behind. For the left loop with current I<sub>1</sub>, we get the following equation:</p>
<p>&nbsp;</p>
<p>$$28 - R_1I_1 - R_2(I_1+I_2) =28 - 4I_1 - 2(I_1+I_2) = 0$$</p>
<p>&nbsp;</p>
<p>Notice that the middle term of the equation uses the sum of mesh currents I<sub>1</sub> and I<sub>2</sub> as the current through resistor R<sub>2</sub>. This is because mesh currents I<sub>1</sub> and I<sub>2</sub> are going in the same direction through R<sub>2</sub>&nbsp;and thus complement each other.&nbsp;</p>
<p>We can distribute the coefficient of 2 to the I<sub>1</sub> and I<sub>2</sub> terms and then combine the I<sub>1</sub> terms to simplify the equation as:</p>
<p>&nbsp;</p>
<p>$$28 - 4I_1 - 2I_1 - 2I_2 = 28 - 6I_1 - 2I_2 = 0$$</p>
<p>&nbsp;</p>
<p>At this time, we have one equation with two unknowns. To be able to solve for two unknown mesh currents, we must have two equations.&nbsp;</p>
<p>Now let&rsquo;s repeat the process for the right loop of the circuit with current I<sub>2</sub>. This will provide us with another KVL equation. With two equations and only two unknown currents, we can solve for the currents. As a creature of habit, we&rsquo;ll again start at the lower-left corner of the right loop and trace clockwise:</p>
<p>&nbsp;</p>
<p>$$R_2(I_1+I_2) +R_3I_2 -7 = 2(I_1+I_2) + 1I_2 - 7 = 0$$</p>
<p>&nbsp;</p>
<p>Simplifying the equation as before, we end up with:</p>
<p>&nbsp;</p>
<p>$$2I_1 + 2 I_2 + 1I_2 - 7 = 2I_1 + 3I_2 - 7 = 0$$</p>
<p>&nbsp;</p>
<h4>Step 4: Solve the Simultaneous Equations for the Unknown Currents</h4>
<p>Now, with two equations, we can use any of several methods to mathematically solve for the unknown currents I<sub>1</sub> and I<sub>2</sub>. First, let&rsquo;s rearrange the two equations for an easier solution:</p>
<p>&nbsp;</p>
<p>$$6I_1 + 2I_2 = 28$$</p>
<p>&nbsp;</p>
<p>$$2I_1 + 3I_2 = 7$$</p>
<p>&nbsp;</p>
<p>Now, solving for the currents, we get:</p>
<p>&nbsp;</p>
<p>$$I_1 = 5 \text{ A}$$</p>
<p>&nbsp;</p>
<p>$$I_2 = -1 \text{ A}$$</p>
<p>&nbsp;</p>
<h4>Step 5: Redraw the Mesh Currents and Determine the Branch Currents</h4>
<p>Knowing that these solutions are values for mesh currents, not branch currents, we must go back to our diagram to see how they fit together to give currents through all components (Figure 4).</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_4._Circuit_with_calculated_mesh_current_values_.jpg" /></p>
<h5><em><strong>Figure 4.</strong> Circuit with calculated mesh current values.</em></h5>
<p>&nbsp;</p>
<p>The solution of -1 A for I<sub>2</sub> means that our initially assumed current direction was incorrect. In actuality, I<sub>2</sub> flows in a clockwise direction at a value of +1 A. With this correction, let&rsquo;s redraw our circuit by changing the current flow direction for I<sub>2</sub> and the voltage drop for resistor R<sub>3</sub>, as shown in Figure 5.</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_5._Circuit_with_corrected_mesh_current_direction_for_I2_.jpg" /></p>
<h5><em><strong>Figure 5.</strong> Circuit with corrected mesh current direction for I<sub>2</sub>.</em></h5>
<p>&nbsp;</p>
<p>Now, from these mesh currents, we can determine the branch currents for our circuit. We can easily see that the current through B<sub>1</sub> and R<sub>1</sub> is 5 A since only mesh current I<sub>1</sub> passes through those two circuit components. Similarly, a current of 1 A is flowing through R<sub>3</sub> and into B<sub>2</sub>.</p>
<p>After that, you might question, what about the R<sub>2</sub>? It has two mesh currents passing through it. Mesh current I<sub>1</sub> is 5 A going &ldquo;down&rdquo; through R<sub>2</sub>, while mesh current I<sub>2</sub> is 1 A going &ldquo;up&rdquo; through R<sub>2</sub>.&nbsp;</p>
<p>To determine the actual current through R<sub>2</sub>, we must see how the mesh currents I<sub>1</sub> and I<sub>2</sub> interact. In this case, they&rsquo;re in opposition&mdash;flowing in opposite directions. We can algebraically add them to arrive at a final value:</p>
<p>&nbsp;</p>
<p>$$I_{R2} = I_1 - I_2 = 5 - 1 = 4 \text{ A}$$</p>
<p>&nbsp;</p>
<p>The current through R<sub>2</sub> must be a value of 4 A, going &ldquo;down.&rdquo; All of the branch currents are shown in Figure 6.</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_6._Circuit_with_calculated_branch_currents_.jpg" /></p>
<h5><em><strong>Figure 6. </strong>Circuit with calculated branch currents.</em></h5>
<p>&nbsp;</p>
<h4>Step 6: Calculate the Voltage Drops</h4>
<p>Now that we know all of the branch currents, we can use Ohm&rsquo;s law to calculate the unknown voltage drops across the resistors in the circuit.</p>
<p>&nbsp;</p>
<p>$$V_{R1} = I_{R1}R_1 = I_1R_1 = 5\cdot4 = 20 \text{ V}$$</p>
<p>&nbsp;</p>
<p>$$V_{R2} = I_{R2}R_2 = (I_1-I_2)R_1 = (5-1)\cdot 2 = 8 \text{ V}$$</p>
<p>&nbsp;</p>
<p>$$V_{R3} = I_{R3}R_3 = I_2R_3 = 1\cdot1 = 1 \text{ V}$$</p>
<p>&nbsp;</p>
<p>We could check our results by returning to Kirchhoff&rsquo;s voltage law for our two loops:</p>
<p>&nbsp;</p>
<p>$$V_{B1} - V_{R1} - V_{R2} = 28 - 20 - 8 = 0 \text{ V}$$</p>
<p>&nbsp;</p>
<p>$$V_{R2} - V_{R3} - V_{B2} = 8 - 1 - 7 = 0 \text{ V}$$</p>
<p>&nbsp;</p>
<h3>Advantages of Mesh Current vs Branch Current Methods</h3>
<p>The primary advantage of mesh current analysis is that it generally allows for the solution of a large network with fewer unknown values and fewer simultaneous equations. In our example circuit, it requires three equations to solve the branch current method and only two equations using the mesh current method. This advantage can be more pronounced in networks with increased complexity, such as the circuit shown in Figure 7.</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_7._Example_circuit_with_increased_complexity_.jpg" /></p>
<h5><em><strong>Figure 7.</strong> Example circuit with increased complexity.</em></h5>
<p>&nbsp;</p>
<p>To solve this network using branch currents, we&rsquo;d have to establish five variables to account for each and every unique current in the circuit (I<sub>1</sub> through I<sub>5</sub>), as demonstrated in Figure 8.</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_8._Complex_circuit_setup_for_Branch_Current_analysis_.jpg" /></p>
<h5><em><strong>Figure 8.</strong> Complex circuit setup for branch current analysis.</em></h5>
<p>&nbsp;</p>
<p>This would require five equations for the solution in the form of two KCL equations at the nodes and three KVL equations for the loops:</p>
<p>For the circuit of Figure 8, our five equations would be:</p>
<p>&nbsp;</p>
<p>$$I_1 - I_2 - I_3 = 0$$</p>
<h5><em>KCL at node 1</em></h5>
<p>&nbsp;</p>
<p>$$I_3 - I_4 - I_5 = 0$$</p>
<h5><em>KCL at node 2</em></h5>
<p>&nbsp;</p>
<p>$$V_{B1} - I_1R_1 - I_2R_2 = 0$$</p>
<h5><em>KVL of the left loop</em></h5>
<p>&nbsp;</p>
<p>$$I_2R_2 - I_3R_3 - I_4R_4 = 0$$</p>
<h5><em>KVL of the middle loop</em></h5>
<p>&nbsp;</p>
<p>$$I_4R_4 - I_5R_5 - V_{B2} = 0$$</p>
<h5><em>KVL of the right loop</em></h5>
<p>&nbsp;</p>
<p>All in all, if you have nothing better to do with your time than to solve for five unknown variables with five equations, you might not mind using the branch current method of analysis for this circuit. However, for those of us who have better things to do with our time, the mesh current method is a whole lot easier, requiring only three unknowns and three equations to solve, as illustrated in Figure 9.</p>
<p>&nbsp;</p>
<p><img src="/uploads/thumbnails/Figure_9._Complex_circuit_setup_for_Mesh_Current_analysis_.jpg" /></p>
<h5><em><strong>Figure 9. </strong>Complex circuit setup for mesh current analysis.</em></h5>
<p>&nbsp;</p>
<p>Our three KVL equations using the mesh current method are:</p>
<p>&nbsp;</p>
<p>$$V_{B1} - I_1R_1 - (I_1-I_2)R_2 = 0$$</p>
<h5><em>KVL of the left loop</em></h5>
<p>&nbsp;</p>
<p>$$(I_2-I_1)R_2 - I_2R_3 - (I_2-I_3)R_4= 0$$</p>
<h5><em>KVL of the middle loop</em></h5>
<p>&nbsp;</p>
<p>$$(I_3-I_2)R_4 - I_3R_5 - V_{B2}= 0$$</p>
<h5><em>KVL of the right loop</em></h5>
<p>&nbsp;</p>
<p>With fewer equations to work with, this method has a clear advantage, especially when performing a simultaneous equation solution by hand without a calculator or computer.</p>

<h3>Review of the Mesh Current Method</h3>
<ul>
<li>The mesh current method is a network analysis technique in which mesh (or loop) currents are assigned arbitrarily, and then Kirchhoff&rsquo;s voltage law and Ohm&rsquo;s law are applied systematically to solve for all unknown currents and voltages.</li>
<li><strong>Step 1: </strong>Identify and label the current loops</li>
<li><strong>Step 2: </strong>Label the voltage drop polarities</li>
<li><strong>Step 3:</strong> Apply Kirchhoff&rsquo;s voltage law to each loop</li>
<li><strong>Step 4:</strong> Solve the simultaneous equations for the unknown currents</li>
<li><strong>Step 5: </strong>Redraw the mesh currents and determine the branch currents</li>
<li><strong>Step 6:</strong> Calculate the voltage drops using Ohm&rsquo;s law</li>
<li>The mesh current method typically provides an easier solution using fewer simultaneous equations than the branch current method&nbsp;</li>
</ul>
