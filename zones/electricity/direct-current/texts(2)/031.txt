# Hand Calculator Use
<p>To enter numbers in scientific notation into a hand calculator, there is usually a button marked &ldquo;E&rdquo; or &ldquo;EE&rdquo; used to enter the correct power of ten. For example, to enter the mass of a proton in grams (1.67 x 10<sup>-24</sup> grams) into a hand calculator, I would enter the following keystrokes:</p>
<pre>
[1] [.] [6] [7] [EE] [2] [4] [+/-]</pre>
<p>The [+/-] keystroke changes the sign of the power (24) into a -24. Some calculators allow the use of the subtraction key [-] to do this, but I prefer the &ldquo;change sign&rdquo; [+/-] key because its more consistent with the use of that key in other contexts.</p>
<p>If I wanted to enter a negative number in scientific notation into a hand calculator, I would have to be careful how I used the [+/-] key, lest I change the sign of the power and not the significant digit value. Pay attention to this example:</p>
<p>Number to be entered: -3.221 x 10<sup>-15</sup>:</p>
<pre>
[3] [.] [2] [2] [1] [+/-] [EE] [1] [5] [+/-]</pre>
<p>The first [+/-] keystroke changes the entry from 3.221 to -3.221; the second [+/-] keystroke changes the power from 15 to -15.</p>

## Metric and Scientific Display Modes on a Handheld Calculator
<p>Displaying metric and scientific notation on a hand calculator is a different matter. It involves changing the display option from the normal &ldquo;fixed&rdquo; decimal point mode to the &ldquo;scientific&rdquo; or &ldquo;engineering&rdquo; mode. Your calculator manual will tell you how to set each display mode.</p>
<p>These display modes tell the calculator how to represent any number on the numerical readout. The actual value of the number is not affected in any way by the choice of display modes&mdash;only how the number appears to the calculator user.</p>
<p>Likewise, the procedure for entering numbers into the calculator does not change with different display modes either. Powers of ten are usually represented by a pair of digits in the upper-right hand corner of the display, and are visible only in the &ldquo;scientific&rdquo; and &ldquo;engineering&rdquo; modes.</p>
## Scientific vs. Engineering Display Modes
<p>The difference between &ldquo;scientific&rdquo; and &ldquo;engineering&rdquo; display modes is the difference between scientific and metric notation. In &ldquo;scientific&rdquo; mode, the power-of-ten display is set so that the main number on the display is always a value between 1 and 10 (or -1 and -10 for negative numbers).</p>
<p>In &ldquo;engineering&rdquo; mode, the powers-of-ten are set to display in multiples of 3, to represent the major metric prefixes. All the user has to do is memorize a few prefix/power combinations, and his or her calculator will be &ldquo;speaking&rdquo; metric!</p>
<pre>
POWER METRIC PREFIX&#10; 12 ......... Tera (T) &#10; 9 .......... Giga (G) &#10; 6 .......... Mega (M) &#10; 3 .......... Kilo (k) &#10; 0 .......... UNITS (plain) &#10; -3 ......... milli (m) &#10; -6 ......... micro (u)&#10; -9 ......... nano (n)&#10; -12 ........ pico (p)&#10;</pre>

<strong>REVIEW:</strong></p>
<ul>
<li>Use the [EE] key to enter powers of ten.</li>
<li>Use &ldquo;scientific&rdquo; or &ldquo;engineering&rdquo; to display powers of ten, in scientific or metric notation, respectively.</li>
</ul>
