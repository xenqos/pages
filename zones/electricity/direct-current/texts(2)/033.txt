# What is the Difference Between Series and Parallel Circuits?
<p><img src="/uploads/thumbnails/Chapter_5_-_Series_And_Parallel_Circuits.jpg" /></p>
<h3 class="text-lg lg:text-2xl font-bold mb-3">What are Series and Parallel Circuits?</h3>
<p><p>In a series circuit, all components are connected end-to-end to form a single path for current flow. In a parallel circuit, all components are connected across each other with exactly two electrically common nodes with the same voltage across each component.</p></p>
<p>Circuits consisting of just one <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-11/battery-construction/" target="_blank">battery</a> and one load <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-1/resistance/" target="_blank">resistance</a> are very simple to analyze, but they are not often found in practical applications. Usually, we find circuits where more than two components are connected together.&nbsp;</p>
<p>There are two fundamental ways in which to connect more than two circuit components: <a href="https://www.allaboutcircuits.com/video-lectures/series-circuits-part-1/" target="_blank">series</a> and <a href="https://www.allaboutcircuits.com/video-lectures/parallel-circuits/" target="_blank">parallel</a>. These two basic connection methods can be combined to create more complex <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-7/what-is-a-series-parallel-circuit/" target="_blank">series-parallel circuits</a>.</p>

## What is a Series Connection?
<p>The definition of a <a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-5/simple-series-circuits/" target="_blank">series circuit</a> is a circuit where the components are connected end-to-end in a line as illustrated in Figure 1.</p>

<p><img src="/uploads/articles/Figure_1._Series_connection_of_resistors__.jpg" /></p>
#### <em><strong>Figure 1.</strong> Series connection of resistors</em>

<p>These series <a href="https://www.allaboutcircuits.com/video-lectures/electronic-components-resistors/" target="_blank">resistors</a> form a single path through which current can flow.</p>

## Introduction to Series Circuits&mdash;A Series Circuit Example
<p>Now, let&rsquo;s examine an example of a series circuit as shown in Figure 2:</p>

<p><img src="/uploads/articles/Figure_2._Series_circuit_.jpg" /></p>
#### <em><strong>Figure 2. </strong>An example of a series circuit.</em>

<p>Here, we have three resistors (labeled R<sub>1</sub>, R<sub>2</sub>, and R<sub>3</sub>) connected in a long chain from one battery terminal to the other. Each resistor in a series circuit shares one electrical node with its nearest neighbor.</p>

<blockquote>
<p>Note: The subscript labels&mdash;those little numbers to the lower-right of the letter &ldquo;R&rdquo;&mdash;are unrelated to the resistor values in ohms. They serve only to identify one resistor from another.</p>
</blockquote>

<p>A series circuit&rsquo;s defining characteristic is that all components in a series circuit have the same current flowing through them. There is only one path for the current to flow. In the circuit from Figure 2, the current (I) flows clockwise to complete a full loop from the positive battery terminal back to the negative terminal and then through the battery following the path 1&ndash;2&ndash;3&ndash;4&ndash;1.</p>

## What Is a Parallel Connection?
<p>The definition of a&nbsp;<a href="https://www.allaboutcircuits.com/textbook/direct-current/chpt-5/simple-parallel-circuits/" target="_blank">parallel circuit</a> is a circuit where all components are connected across each other&rsquo;s leads as shown in Figure 3.</p>

<p><img src="/uploads/articles/Figure_3._Parallel_connection_of_resistors__.jpg" /></p>
#### <em><strong>Figure 3.</strong> An example of a parallel connection of resistors.</em>

<p>In a purely parallel circuit, there are never more than two sets of electrically common points, no matter how many components are connected. There are many paths for current flow, but only one voltage across all components.</p>

## Introduction to Parallel Circuits&mdash;A Parallel Circuit Example
<p>Let&rsquo;s look at an example of a parallel circuit as shown in Figure 4.</p>

<p><img src="/uploads/articles/Figure_4._Parallel_circuit_.jpg" /></p>
#### <em><strong>Figure 4.&nbsp;</strong>Example of a parallel circuit.</em>

<p>Again, we have three resistors, but this time there are three loops for the current to flow from the positive battery terminal back to the negative terminal:</p>
<ol>
<li>1&ndash;2&ndash;7&ndash;8&ndash;1</li>
<li>1&ndash;2&ndash;3&ndash;6&ndash;7&ndash;8&ndash;1</li>
<li>1&ndash;2&ndash;3&ndash;4&ndash;5&ndash;6&ndash;7&ndash;8&ndash;1</li>
</ol>
<p>Each individual path through R<sub>1</sub>, R<sub>2</sub>, and R<sub>3</sub> (2&ndash;7, 3&ndash;6, and 4&ndash;5) is called a branch.</p>
<p>A parallel circuit&rsquo;s defining characteristic is that all components are connected between the same set of electrically common points. Looking at the schematic diagram from Figure 4, we see that points 1, 2, 3, and 4 are all electrically common; so are points 5, 6, 7, and 8. All of the resistors, as well as the battery, are connected between these two sets of points. This means that the same voltage (V) is dropped across all components in a parallel circuit.</p>

## Series vs&nbsp;Parallel Circuit Review:
<ul>
<li>In a series circuit, all components are connected end-to-end, forming a single path for current flow.</li>
<li>In a parallel circuit, all components are connected across each other, forming exactly two sets of electrically common points.</li>
<li>A &ldquo;branch&rdquo; in a parallel circuit is a path for electric current formed by one of the load components (such as a resistor).</li>
</ul>
